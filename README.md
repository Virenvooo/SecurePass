# SecurePass - Modern Password Generator

A beautiful, secure, and easy-to-use password generator built with **customtkinter**.

![SecurePass]([https://i.imgur.com/placeholder.png](https://i.imgur.com/ZFulVl8.png))

### Features
- Real-time strength indicator
- Live preview while changing options
- Fully rounded modern UI
- Copy to clipboard with feedback
- Fixed beautiful layout (no resizing issues)
- Auto-generates on every change

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/Virenvooo/SecurePass.git
cd SecurePass

# 2. Install requirements
pip install -r requirements.txt

# 3. Run it
python app.py
